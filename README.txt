Pour compiler le programme, dans le terminal allez dans le dossier qui contient le Makefile, entrez la commande make.
LA LIBRAIRIE GRAPHIQUE MLV EST NECESSAIRE POUR COMPILER LE PROGRAMME.
Ensuite, toujours dans le terminal, pour éxécuter le programme, faites la commande ./voyageur LARGEUR LONGUEUR NBR_VILLE où largeur, longueur et nbr_ville sont des entiers strictement positifs.
Le programme vous affichera les villes et un chemin qui évoluera pour devenir de plus en plus court.
